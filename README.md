## 👩🏼‍🤝‍👩🏼 프로젝트 소개
팀 내의 효율적인 협업과 커뮤니케이션을 돕기 위해 제작한 홈페이지 

## 🔊 기능 소개
- 회원가입, 로그인 (이메일, 카카오 기반)
- 워크스페이스, Todo 생성/삭제/수정
- 실시간 채팅 (텍스트, 이모티콘, 코드 블럭, 파일 전송)
- 회원 정보 수정, 탈퇴, 로그아웃
